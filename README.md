# ✨ About GoalSync

GoalSync is a modern **Enterprise Goal Management Platform** designed to help organizations, teams, managers, and employees efficiently manage goals, monitor performance, and improve productivity.

Built with the **MERN Stack**, GoalSync provides a scalable, secure, and responsive system for enterprise-level workflow and performance management.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/maurya-78/GoalSync.git
```

---

# 🌐 Frontend Setup

```bash
cd GoalSync/Frontend

npm install

npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```
---

# ⚙️ Backend Setup

```bash
cd GoalSync/Backend

npm install

npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```
---

# 🔌 Environment Variables

Create `.env` inside Backend folder:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:3000
```

# 🔐 API Endpoints

## Authentication APIs

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/auth/register | Register User |
| POST | /api/auth/login | Login User |
| GET | /api/auth/me | Current User |
| POST | /api/auth/forgot-password | Forgot Password |

---

## Goal APIs

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/goals | Get Goals |
| POST | /api/goals | Create Goal |
| PUT | /api/goals/:id | Update Goal |
| DELETE | /api/goals/:id | Delete Goal |
